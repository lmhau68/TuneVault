# TuneVault

TuneVault là bài tập lớn môn C# and .NET Development.

Ứng dụng mô phỏng nền tảng nghe nhạc và xem video trực tuyến, có giao diện tham khảo kiểu Spotify.

## Công nghệ sử dụng

### Frontend
- React
- TypeScript
- Vite
- React Router
- Axios
- SignalR Client

### Backend
- ASP.NET Core 8 Web API
- Clean Architecture
- Dapper
- SQL Server
- JWT Authentication
- Swagger/OpenAPI
- SignalR
